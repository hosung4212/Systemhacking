# Dreamhack System Hacking Write-ups

## Overview
이 레포지토리는 :contentReference[oaicite:0]{index=0}의 **시스템 해킹(System Hacking) 문제 풀이(write-up)**를 정리한 공간입니다.

각 문제에 대해 단순한 정답 코드 나열이 아니라,  
- 취약점이 발생하는 원인  
- 공격 흐름과 exploit 구성 과정  
- 보호 기법이 적용된 환경에서의 동작 방식

- GDB 기반 디버깅 및 exploit 작성

문제별로 독립적인 풀이가 정리되어 있으며,  
시스템 해킹 전반에 대한 이해를 목표로 합니다.
